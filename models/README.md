# Mongoose Models

Database models for MongoDB.

## Models

- `User.ts` - User accounts with roles
- `Session.ts` - DB-backed sessions
- `Lawyer.ts` - Lawyer profiles
- `Category.ts` - Post categories
- `Post.ts` - Blog posts
- `Comment.ts` - Post comments
- `AskLawyer.ts` - Private lawyer inquiries
- `Lead.ts` - Lead submissions
- `Download.ts` - Downloadable resources
- `Video.ts` - Video library items
- `AnalyticsEvent.ts` - Analytics tracking

Will be created in the next phase.
